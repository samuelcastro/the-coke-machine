# Running and Testing

### `yarn`

  Run `yarn` to install all necessary dependencies.

### `yarn start`

  Use `yarn start` to start the app

### `yarn test`

  Run all tests with `yarn test`, I'm using [Jest](https://jestjs.io/), which is amazing.
